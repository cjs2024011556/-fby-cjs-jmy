MotoEngine Workbench

摩托车发动机维修智能体

这是一个把 维修手册切分、向量检索、章节 Workflow、SSE 流式输出、会话历史和数据库管理 串起来的完整项目。

核心能力
  • 按章节拆分维修手册并生成向量库
  • RAG 统一问答和章节级 Workflow 指导
  • 聊天历史持久化与 SSE 流式输出
  • 独立数据库可视化页面，支持增删改查
主要目录
  • motoengine/func/ 核心逻辑
  • motoengine/models/ 数据模型
  • motoengine/route/ API 路由
  • motoengine/templates/ 前端页面
